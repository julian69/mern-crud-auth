## Auth & Task Manager (IN PROGRESS)

### Description
Task Manager is a simple application that allows you to create, edit and delete tasks. It also has a simple authentication system.

- React Context API

- Express

### Scripts
Run the server

```bash
$ cd server
```
```bash
$ npm run dev
```

Run the client

```bash
$ cd client
```
```bash
$ npm run dev
```
